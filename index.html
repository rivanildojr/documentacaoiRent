AvaliacaoOfertas

Up 
put /avaliacaoOfertas/:id
atualiza a avaliação de uma oferta (atualizaAvaliacaoOferta)
rota que atualiza uma avaliação pelo seu id

Path parameters

id (required)
Path Parameter — id da avaliação

Request body

json json_2 (required)
Body Parameter — é preciso enviar o seguinte campo no arquivo JSON para fazer a atualização de uma avaliação

Return type

AvaliacaoOferta

Example data

Content-Type: application/json
{
  "updated_at" : "2019-06-25 21:00:09-03",
  "user_id" : 44,
  "created_at" : "2019-06-17 21:17:36-03",
  "id" : 21,
  "oferta_id" : 12,
  "nota" : 4
}

Responses

200

operação realizada com sucesso AvaliacaoOferta
Up 
post /avaliacaoOfertas
criar nova avaliação de oferta (createAvaliacaoOfertas)
rota que adiciona uma nova avaliação a uma oferta

Request body

json json_1 (required)
Body Parameter — são precisos os seguintes campos no arquivo JSON para fazer a avaliação de uma oferta

Return type

inline_response_201

Example data

Content-Type: application/json
{
  "message" : "Avaliação realizada!"
}

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

201

OK! Serão retornados os seguintes campos em formato json, caso a avaliação tenha sido registrada inline_response_201

409

Conflict! Já existe uma avaliação para esse usuário em determinada oferta. Se quiser atualizar, use o método para atualizar em vez desse inline_response_201
Up 
delete /avaliacaoOfertas/:oferta_id
deleta avaliação pelo seu id (deleteAvaliacaoOferta)
rota que deleta uma avaliação de oferta pelo id da avaliacaoOferta

Return type

inline_response_200_2

Example data

Content-Type: application/json
{
  "message" : "message: Avaliação de oferta removida"
}

Responses

200

OK! Avaliação deletada com sucesso inline_response_200_2

404

NOT FOUND! Avaliação não encontrada inline_response_404
Up 
get /avaliacaoOfertas
retorna todas as avaliações de ofertas existentes (getAllAvaliacaoOferta)
rota que retorna todas as avaliações de ofertas armazenadas

Return type

AvaliacaoOferta

Example data

Content-Type: application/json
{
  "updated_at" : "2019-06-25 21:00:09-03",
  "user_id" : 44,
  "created_at" : "2019-06-17 21:17:36-03",
  "id" : 21,
  "oferta_id" : 12,
  "nota" : 4
}

Responses

200

OK! Retornou todas as avaliações em formato JSON AvaliacaoOferta
Up 
get /avaliacaoOfertas/:oferta_id
retorna média de avaliações de uma dada oferta (mediaAvaliacaoOfertas)
rota que retorna a média das avaliações de uma oferta através de seu id

Path parameters

oferta_id (required)
Path Parameter — id da oferta que se deseja obter média de avaliações

Return type

inline_response_200_1

Example data

Content-Type: application/json
{
  "avg" : 4.0
}

Responses

200

média retornada com sucesso inline_response_200_1

Integer id da oferta cuja avaliação pertence
user_id
Integer id do usuário cuja avaliação pertence
nota
Integer nota da avaliação(de 0 à 5 no formato estrela)
created_at (optional)
Object "timestamp:" informe de quando a avaliação foi criada
updated_at (optional)
Object "timestamp:" informe de quando a avaliação foi alterada

inline_response_200 - Up

type (optional)
String tipo do token. Estamos usando token do tipo "bearer"
token (optional)
String sequência de caracteres que representa o token usado para autenticação
refreshToken (optional)
String token de atualização. Não é muito usado e geralmente vem com valor null

BigDecimal "avg:" 4.0000000000000000 é um exemplo de retorno válido, com média de avaliações igual a 4
