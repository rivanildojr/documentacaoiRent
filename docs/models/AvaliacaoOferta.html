AvaliacaoOferta

  • id : Integer
    id da avaliação
  • oferta_id : Integer
    id da oferta cuja avaliação pertence
  • user_id : Integer
    id do usuário cuja avaliação pertence
  • nota : Integer
    nota da avaliação(de 0 à 5 no formato estrela)
  • created_at : Object
    \\\"timestamp:\" informe de quando a avaliação foi criada
  • updated_at : Object
    \\\"timestamp:\" informe de quando a avaliação foi alterada
