This is the API

atualizaAvaliacaoOferta

rota que atualiza uma avaliação pelo seu id

URL

https://localhost/avaliacaoOfertas/:id

HTTP Method

PUT

Response Type

AvaliacaoOferta

Parameters

  • Path: id Integer

    id da avaliação

  • Body: json Json_2(Json_2)

    é preciso enviar o seguinte campo no arquivo JSON para fazer a atualização de uma avaliação

createAvaliacaoOfertas

rota que adiciona uma nova avaliação a uma oferta

URL

https://localhost/avaliacaoOfertas

HTTP Method

POST

Response Type

inline_response_201

Parameters

  • Body: json Json_1(Json_1)

    são precisos os seguintes campos no arquivo JSON para fazer a avaliação de uma oferta

deleteAvaliacaoOferta

rota que deleta uma avaliação de oferta pelo id da avaliacaoOferta

URL

https://localhost/avaliacaoOfertas/:oferta_id

HTTP Method

DELETE

Response Type

inline_response_200_2

Parameters

getAllAvaliacaoOferta

rota que retorna todas as avaliações de ofertas armazenadas

URL

https://localhost/avaliacaoOfertas

HTTP Method

GET

Response Type

AvaliacaoOferta

Parameters

mediaAvaliacaoOfertas
