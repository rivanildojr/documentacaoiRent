This is the API

destroy

URL

https://localhost/users/:username

HTTP Method

DELETE

Response Type

Parameters

  • Path: username String

    username do usuario que será removido

index

rota na qual retorna dos os usuarios cadastrado no banco de dados

URL

https://localhost/users

HTTP Method

GET

Response Type

List

Parameters

show

rota que retorna uma usuario

URL

https://localhost/users/:username

HTTP Method

GET

Response Type

User

Parameters

  • Path: username String

    username do usuario que será atualizado

store

rota que retorna criar um usuario no banco de dados

URL

https://localhost/users

HTTP Method

POST

Response Type

Parameters

  • Body: username User(User)

    username que será atualizado

URL

https://localhost/users/:username

HTTP Method

PUT

Response Type

User

Parameters

  • Path: username String

    username do usuario que será atualizado

  • Body: username User(User)

    username que será atualizado
